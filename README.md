## افزودن فایل فونت‌ها
در این برنامه از قلم وزیرمتن برای متن کتاب دیجیتالی استفاده می‌شود؛ اما این قلم‌ها باید به‌صورت دستی به فایل کتاب اضافه شوند. برای این کار این مرحله‌ها را دنبال کنید:
1. بسته‌ی فونت‌های وزیرمتن را از [این صفحه](https://rastikerdar.github.io/vazirmatn/) دریافت کنید.
2. فایل دریافتی را از حالت فشرده خارج کنید.
3. در بخش «تنظیمات کتاب»  روی گزینه‌ی «انتخاب فایل فونت‌ها» بزنید.
4. به مسیری که پوشه‌ی فونت‌ها را در آن ذخیره کردید، بروید.
5. در پوشه‌ی فونت‌ها به این مسیر بروید: `/fonts/ttf/..`
6. از میان فونت‌های دریافتی فقط این سه قلم را انتخاب کنید:
 - `Vazirmatn-Regular.ttf`
 - `Vazirmatn-Bold.ttf`
 - `Vazirmatn-Black.ttf`

## فهرست
بخش فهرست به صورت خودکار به کتاب اضافه می‌شود و نیازی نیست به صورت دستی وارد شود.

## پشتیبانی از markdown
متن کتاب از قالب markdown برای نمایش پشتیبانی می‌کند. برای ایجاد حالت‌هایی مانند ضخیم‌نویسی یا ایرانیک یا صورت‌های دیگر می‌توانید از این الگوها استفاده کنید:
```Markdown
# عنوان سطح ۱
## عنوان سطح ۲
### عنوان سطح ۳
**متن ضخیم**
*متن ایرانیک*
~~متن خط‌خورده~~
`کد درون‌خطی`
> نقل‌قول
- فهرست بی‌شماره
1. فهرست شماره‌دار
[متن پیوند](https://example.com)

```
⚠️ در برخی از برنامه‌های کتابخوان (مانند Moon Reader) استفاده از این الگوها باعث چپ‌چین شدن سطر می‌شود. این مشکل مربوط به خود کتابخوان است.

## پشتیبان‌گیری
برای ذخیره‌ی پروژه‌ی نیمه‌کاره‌ی خود و بازیابی و ویرایش آن در زمان‌های دیگر می‌توانید از بخش «ذخیره و بازیابی پشتیبان» استفاده کنید. با انتخاب گزینه‌ی **ذخیره** فایلی با پسوند `.json` در پوشه دانلود ذخیره می‌شود. برای بازیابی و تکمیل پروژه‌ی خود می‌توانید گزینه‌ی **بارگذاری** را انتخاب کنید و همان فایل `.json` ذخیره‌شده را وارد برنامه کنید.

⚠️ توجه داشته باشید که فونت‌ها بازیابی نمی‌شوند و برای گرفتن خروجی باید بار دیگر فونت‌ها را، با همان روش یادشده در بالا، انتخاب کنید.
