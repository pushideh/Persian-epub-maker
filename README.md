## درباره
این برنامه توسط هوشمص کلاد نوشته شده و هدف آن ساخت کتاب‌های دیجیتالی با فرمت epub است که برای زبان پارسی مناسب‌سازی شده و پیکربندی آن برای حالت راست‌چین تنظیم شده است.

## افزودن فایل فونت‌ها
در این برنامه از فونت وزیرمتن برای استفاده در کتاب دیجیتالی استفاده می‌شود. اما این فونت‌ها باید به‌صورت دستی به فایل کتاب اضافه شوند. برای این کار این مرحله‌ها را دنبال کنید:
1. بسته‌ی فونت‌های وزیرمتن را از [این صفحه](https://rastikerdar.github.io/vazirmatn/) دریافت کنید.
2. فایل دریافتی را از حالت فشرده خارج کنید.
3. در بخش «تنظیمات کتاب» گزینه‌ی «انتخاب فایل فونت‌ها» را انتخاب کنید.
4. به پوشه‌ای که فونت‌ها در آن هستند بروید.
5. از میان فونت‌های دریافتی فقط این سه فونت را انتخاب کنید:
 - `Vazirmatn-Regular.ttf`
 - `Vazirmatn-Bold.ttf`
 - `Vazirmatn-Black.ttf`

## فهرست
بخش فهرست به صورت خودکار به کتاب اضافه می‌شود و نیازی نیست به صورت دستی وارد شود.

## پشتیبانی از markdown
متن کتاب از قالب markdown برای نمایش پشتیبانی می‌کند. برای ایجاد حالت‌هایی مانند ضخیم‌نویسی یا ایرانیک یا صورت‌های دیگر می‌توانید از این الگوها استفاده کنید:
```Markdown
# تیتر سطح ۱
## تیتر سطح ۲
### تیتر سطح ۳
**متن بولد**
*متن ایتالیک*
~~متن خط‌خورده~~
`کد درون‌خطی`
> نقل‌قول
- آیتم فهرست
1. آیتم فهرست شماره‌دار
[متن لینک](https://example.com)

```
